# Nand2Tetris - Hack Assembly Language Programs

## Overview

This repository contains my personal implementations of the Hack Assembly programming assignments from the **Nand2Tetris** course.

The programs were written by me while following the course curriculum and implementing the required specifications using the Hack Assembly language. These assignments provided hands-on experience with low-level programming, memory manipulation, branching, loops, arithmetic operations, and interacting with the Hack computer platform.

## Programs

### Project 4

### Mult.asm
- Performs multiplication using repeated addition.
- Demonstrates loops, counters, and arithmetic operations.

### Fill.asm
- Reads keyboard input.
- Fills or clears the Hack computer screen depending on whether a key is pressed.
- Demonstrates memory-mapped I/O and conditional branching.

---

### Project 6

This project contains additional Hack Assembly programs demonstrating:

- Arithmetic operations
- Conditional execution
- Loop construction
- Memory access
- Program control
- Basic graphics using the Hack computer screen

## Learning Outcomes

Through these assignments I gained practical experience in:

- Understanding how assembly language interacts directly with computer hardware.
- Implementing algorithms without the support of high-level programming constructs.
- Managing memory manually.
- Building a deeper understanding of computer architecture and instruction execution.
